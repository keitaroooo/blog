### Terraform で管理しているもの

`infra/terraform/netlify/` に配置。Netlify Personal Access Token が必要（Bitwarden の "Netlify Token"）。

- **ビルド設定** (`netlify_site_build_settings`): コマンド、publish dir、base dir

```fish
source infra/load_tokens.fish
cd infra/terraform/netlify
terraform init && terraform plan && terraform apply
```

import ブロックは `main.tf` に記載済み。既に state に取り込まれたリソースは no-op になる。

### Terraform で管理できないもの

- **Legacy prerendering 無効化**: `infra/scripts/disable-prerendering.sh` で API 直叩き
- **Dependency management の Node.js バージョン**: `node/app/.node-version` で指定（Netlify が自動検出）

```fish
source infra/load_tokens.fish
./infra/scripts/disable-prerendering.sh
```
